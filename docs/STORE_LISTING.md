### 完整描述
Tales Runner UMT Map Manager 係一款專為《跑 Online / Tales Runner》玩家設計嘅地圖管理工具。

你可以輕鬆咁從 Google Sheets 地圖資料庫中瀏覽、安裝及管理玩家自製嘅 UMT 地圖。

**主要功能：**
📋 從 Google Sheets 自動讀取地圖資料庫，顯示所有可用地圖
🔍 以關鍵字搜尋地圖，篩選已安裝 / 未安裝狀態
📥 一鍵安裝地圖 — 直接從 Google Drive 下載 .upk 檔案
🔄 停用 / 啟用地圖 — 移至 _deactivated 資料夾而不刪除
🗑️ 安全移除地圖 — 根據安裝記錄安全刪除
☑️ 批次操作 — 勾選多個地圖一次過安裝、移除、啟用或停用
🖼️ 圖片預覽與幻燈片 — 支援點擊放大檢視
🎬 YouTube 影片嵌入 — 地圖介紹影片直接播放
📊 自訂顯示欄位及順序
🌐 多語言支援（繁體中文 / English / 한국어）

地圖資料庫由 幻紫OAO、puihong62871 及 TaiwanPro 共同建立維護。

## English Description

### Short Description (100 chars)
Map manager for Tales Runner (跑 Online). One-click install, deactivate, remove UMT maps with image preview and YouTube embed.

### Full Description
Tales Runner UMT Map Manager is a desktop tool designed for Tales Runner (跑 Online) players to manage player-created UMT maps.

Browse, install, and manage maps from the community Google Sheets database with ease.

**Features:**
📋 Auto-load map database from Google Sheets
🔍 Search maps by keyword, filter by install status
📥 One-click install — download .upk files directly from Google Drive
🔄 Deactivate / Activate maps — move to _deactivated folder without deleting
🗑️ Safe remove — delete maps based on install records
☑️ Bulk operations — select multiple maps for batch install, uninstall, activate, or deactivate
🖼️ Image preview & slideshow with click-to-zoom
🎬 YouTube video embed — watch map intro videos in-app
📊 Customizable visible columns and order
🌐 Multi-language support (Traditional Chinese / English / 한국어)

Map database collaboratively built and maintained by 幻紫OAO, puihong62871, and TaiwanPro.

Want to submit a new map? Fill out this form: https://docs.google.com/forms/d/e/1FAIpQLScLfPEDOoMfQj9bKD6E0JB-YNDS-HN2YCmUu323kz312acwFQ/viewform
